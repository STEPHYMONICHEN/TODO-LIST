# Django Todo List Application

# Project Overview
This is a Todo List application built using the Django web framework. It is designed to help users manage their daily tasks efficiently with a simple and intuitive interface. Users can add, edit, delete, and mark tasks as completed.

## Features
- Add Tasks: Users can add new tasks to their todo list.
- Edit Tasks: Users can modify existing tasks.
- Delete Tasks: Users can remove tasks from the list.
- Complete Tasks: Users can mark tasks as completed to track their progress.

## Tech Stack
-Backend: Django 3.2
- Database: SQLite (default Django database)
- Frontend: HTML, CSS and Django templates with Bootstrap 5 for styling.

### Prerequisites

You will need to have the following tools installed on your system:
- Python 3.8 or newer
- Pip (Python package installer)
